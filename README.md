# 🕒 MA2 Clock Wallpaper

A customizable **GrandMA2-inspired digital clock** designed to be used as a desktop wallpaper.

Perfect for:

- 🎭 Lighting Designers
- 💡 Lighting Operators
- 🎬 VJs
- 🎵 Show Technicians
- 🎪 Live Event Professionals

## ✨ Features

- GrandMA2 inspired interface
- Real-time clock
- Responsive design
- Lightweight (HTML, CSS & JavaScript)
- Works with Lively Wallpaper
- Easy to customize
- Custom logo or name support

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-user/ma2-clock-wallpaper.git
```

Open `index.html` in your browser.

Or use **Lively Wallpaper** to set it as your desktop background.

## 🎨 Customization

You can easily customize:

- Name
- Logo
- Colors
- Font
- Background
- Glow intensity
